# HR-Analytics-

An interactive Employee Retention Dashboard built with Power BI, analyzing attrition rates, income bands, promotion gaps, work-life balance, and departmental trends across 50K employees.

An interactive HR Analytics Dashboard built with Power BI, providing deep insights into employee attrition, income distribution, promotion gaps, work-life balance, and departmental performance across a workforce of 50,000 employees.

📌 Overview

Employee retention is one of the most critical challenges for any organization. This dashboard provides HR teams and business leaders with a comprehensive, data-driven view of attrition patterns and workforce health metrics — enabling smarter decisions around compensation, promotion policies, and employee well-being.
